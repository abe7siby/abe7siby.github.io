My Projects

Model Identification Using IPCA and NCA


This project was done in three parts. In the first part, dimensionality reduction of Yale faces dataset was done using PCA and obtained an accuracy of 99.3% recognition on the train set. For the second part I Identified model of a complex pipeline network system using IPCA and NCA and found leakages successfully. For the last part, I developed a multivariate calibration NMF model of spectral data and mixture composition in chemometrics and estimated the pure component spectra using Network Component Analysis over denoised spectral data

Air Water Compressibility Effects in An OWC Wave Energy Device


Task assigned was to model the air-water compressibility effects in an Oscillating Water Column wave energy device. The numerical model is formulated using ANSYS FLUENT.Finite element analysis was done to find the pressure variations on the air column inside the column, assuming waves follow STOKES fifth order equation.

Model Resistance Testing


The resistance acting on the model is found by calculating the tension on the hawser that is used to pull the model in the towing tank.

Ship Drawing and Calculations


Drawings such as body plans, half-breadth plans and shear plans of a cargo ship were created of a ship of LOA 170m and then bending moments acting on the actually ship were calculated using this model.
